Tones Music Player - Release APKs v2.5.5
==========================================

This folder contains signed release APKs for Tones Music Player version 2.5.5.

📱 APK Files:
- Tones-universal.apk (21.4 MB) - Universal build supporting all architectures
- Tones-arm64.apk (21.3 MB) - ARM64 for modern Android devices  
- Tones-armeabi.apk (21.3 MB) - ARM32 for older Android devices
- Tones-x86.apk (21.3 MB) - x86 for Intel-based Android devices
- Tones-x86_64.apk (21.3 MB) - x86_64 for 64-bit Intel-based Android devices

🔐 Security:
- All APKs are properly signed with the release keystore
- Verified using APK Signature Scheme v2 and v3
- Ready for distribution and installation

📅 Build Information:
- Build Date: July 28, 2024
- Version: 2.5.5
- Version Code: 255
- Target SDK: 36
- Minimum SDK: 26

✨ New Features in v2.5.5:
- Smooth transition between songs (formerly crossfade)
- Now Playing screen animations with album art transitions
- Search filters (duration, year) for local music
- Thumbnail preloading for smooth album art display
- Local artist profile support - no more errors for local artists
- Various bug fixes and performance improvements

📋 Installation Instructions:
1. Enable "Install from Unknown Sources" in your Android settings
   - Go to Settings > Security > Unknown Sources
   - Or Settings > Apps > Special app access > Install unknown apps
2. Download the appropriate APK for your device architecture
3. Tap on the APK file to install
4. Follow the installation prompts
5. Enjoy Tones Music Player!

💡 Recommendations:
- The universal APK (Tones-universal.apk) is recommended for most users
- It supports all Android device architectures
- Slightly larger file size but guaranteed compatibility

🔧 Troubleshooting:
- If installation fails, ensure "Install from Unknown Sources" is enabled
- For ARM64 devices, you can use either universal or arm64 APK
- For older devices, use the armeabi APK
- For Intel-based devices (tablets, emulators), use x86 or x86_64 APK

📞 Support:
- This is a release build with all optimizations enabled
- Debug features are disabled for better performance
- Report any issues through the app's feedback system
